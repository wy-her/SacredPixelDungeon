# Sacred Pixel Dungeon

An HTML-based project inspired by **Shattered Pixel Dungeon**.

---

## 📌 About

**Sacred Pixel Dungeon** is a fan-made HTML adaptation/mod project based on the open-source game:

👉 Shattered Pixel Dungeon by Evan Debenham
https://github.com/00-Evan/shattered-pixel-dungeon

This project reinterprets the original gameplay in a web-based environment.
